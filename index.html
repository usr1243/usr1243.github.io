Statt eines Agenten, der alles versucht, lässt ein Orchestrator ein Team von Spezialisten parallel arbeiten — jeder lädt nur sein eigenes Wissen. Mein Debugging-Agent geht sogar über 5 Phasen: ein Agent kartografiert den Code, ein Schwarm analysiert, ein zweiter Schwarm behebt die Fehler. Instead of one agent doing everything, an orchestrator runs a team of specialists in parallel — each loading only its own knowledge. My debugging agent even runs 5 phases: one agent maps the code, a swarm analyses, a second swarm fixes the bugs.
